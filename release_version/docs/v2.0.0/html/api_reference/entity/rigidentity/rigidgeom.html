visualize_sdf(pos=None, T=None, color=(1.0, 1.0, 0.3, 1.0), show_axis=False, axis_color=(1.0, 0.0, 0.0, 1.0), axis_length=0.3, show_boundary=False, boundary_color=(0.0, 1.0, 0.0, 0.2))[source]#

Visualizes the signed distance field (SDF) of the rigid geometry in the viewer.

Parameters:

  • pos (Optional[np.ndarray]) – The position of the geometry. If None, the current position of the geometry is used.

  • T (Optional[np.ndarray]) – The pose of the geometry. If None, the current pose of the geometry is used.

  • color (Tuple[float, float, float, float]) – The color of the SDF mesh. Default is (1.0, 1.0, 0.3, 1.0).

  • show_axis (bool) – Whether to show the mesh axis. Default is False.

  • axis_color (Tuple[float, float, float, float]) – The color of the axis. Default is (1.0, 0.0, 0.0, 1.0).

  • axis_length (float) – The length of the axis. Default is 0.3.

  • show_boundary (bool) – Whether to show the boundary of the sdf grid. Default is False.

  • boundary_color (Tuple[float, float, float, float]) – The color of the boundary. Default is (0.0, 1.0, 0.0, 0.2).
